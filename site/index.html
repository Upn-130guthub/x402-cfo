server.ts
import express from 'express';
import { tollbooth } from 'tollbooth';

const app = express();

// One line. That's it.
app.use('/api/premium', tollbooth({
  price: '0.25',
  currency: 'USDC',
  network: 'base',
  recipient: '0xYourWallet',
  mode: 'hosted',
  apiKey: process.env.TOLLBOOTH_API_KEY,
}));

app.get('/api/premium/data', (req, res) => {
  res.json({ data: 'unlocked', receipt: req.tollbooth });
});
How it works

Four steps. Two HTTP requests.

Tollbooth implements the x402 protocol — the HTTP payment standard by Coinbase and Cloudflare.

1
Request
Client hits your protected API endpoint
→
2
402 Challenge
Tollbooth returns payment requirements
→
3
Pay & Retry
Client pays on-chain and retries with proof
→
4
200 + Receipt
Tollbooth verifies, attaches receipt, passes through
